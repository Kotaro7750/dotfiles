# Setup
## zplug
Install zplug by following the instructions on [GitHub](https://github.com/zplug/zplug) page.

Then create below file. The env value may be changed as you like.

```bash
# zsh/path/pre/zplug
export ZPLUG_HOME=~/.zplug/
```
